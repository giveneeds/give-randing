# Threads 인기 게시글 패턴 감사 2026-05-19

본 문서는 기브니즈 Threads 콘텐츠 생성 로직을 만들기 위한 인기 게시글 관찰 기록이다. 목표는 "좋아 보이는 글"을 감으로 따라 하는 것이 아니라, 조회/좋아요/댓글/리포스트/공유와 댓글 반응을 함께 보고 어떤 주제에 어떤 형식이 맞는지 판단하는 것이다.

## 감사 기준

우선순위:
- 좋아요 200개 이상
- 조회수 5만 이상
- 댓글 100개 이상
- 위 조건을 모두 만족하지 않아도 리포스트/공유가 매우 높으면 "저장형 콘텐츠"로 별도 관찰

확인 수준:
- `detail_comment`: 게시글 상세 진입 + 댓글 일부 확인
- `detail_metric`: 게시글 상세 진입 + 조회/반응 수 확인
- `search_card`: 검색 결과 카드 기준 확인
- `embedded_card`: 인기 스레드 안에 인용/임베드된 원본 카드 기준 확인

## 1차 패턴

### 1. 터진 글은 "정보"보다 "쓸모의 약속"이 먼저 온다

나쁜 예:
- 광고비를 쓰는데 문의가 안 올 수 있습니다.

좋은 구조:
- "광고비를 더 쓰기 전에, 오늘 10분 안에 점검할 전환 구멍 4개"

핵심:
- 독자가 이 글을 읽는 이유가 첫 줄에서 보여야 한다.
- "무엇을 알게 되는가"보다 "무엇을 바로 점검/저장/적용할 수 있는가"가 먼저다.

### 2. 공유/리포스트가 높은 글은 댓글형이 아니라 저장형이다

AI 강의, 도구 리스트, 무료 제작법, 리소스 묶음은 댓글보다 공유/리포스트가 높다.

기브니즈 적용:
- "사장님이 이번 주 바로 점검할 5가지"
- "처음 광고하기 전 확인할 도착지 체크 4가지"
- "AI로 내 가게 콘텐츠 소재 뽑는 질문 7개"

### 3. 댓글이 높은 로컬 마케팅 글은 CTA가 강하다

네이버 플레이스 검색 결과에서는 좋아요는 낮아도 답글 비율이 높은 글이 보였다.

공통점:
- "댓글 남기면 점검해줄게"
- "대표님들 전부 해당"
- "이거 아직도 하고 있나요?"처럼 직접 찌름

기브니즈 적용:
- 단순 정보글보다 "댓글에 업종 남기면 예시로 바꿔드릴게요" 같은 검수형 CTA가 더 맞을 수 있다.

### 4. 일반 도구 나열은 반응을 얻지만, 기브니즈 전환과는 거리가 있다

Canva, Buffer, Mailchimp 같은 도구 리스트는 저장 가치는 있지만 타깃이 넓다.

기브니즈 적용:
- 도구 리스트를 그대로 쓰면 안 된다.
- "식당 사장님용", "병원 원장님용", "플레이스 담당자용"처럼 업무 장면으로 좁혀야 한다.

### 5. 우리 이전 초안이 약했던 이유

문제:
- 타깃이 보이지 않았다.
- "왜 지금 봐야 하는지"가 없었다.
- 독자가 당장 얻는 결과가 없었다.
- 댓글/저장/공유 중 어떤 반응을 노리는지 불명확했다.

수정 방향:
- 글마다 목표 반응을 먼저 정한다.
  - 댓글형: 자기 상황을 남기게 한다.
  - 저장형: 체크/도구/질문 리스트를 준다.
  - 공유형: 업종 동료에게 보내고 싶은 명확한 기준을 준다.

## 다음 확인 필요

- 실제 상세 댓글까지 확인한 샘플을 20개로 늘린다.
- 특히 로컬 마케팅/자영업/병원 마케팅 키워드는 검색 결과 반응이 낮으므로, 단순 인기 검색보다 계정/태그 기반 추가 탐색이 필요하다.
- 댓글 100개 이상 샘플은 현재 AI/툴/프롬프트 쪽에 더 많다. 기브니즈 주제에 바로 가져오려면 "댓글형 CTA"를 설계해야 한다.
