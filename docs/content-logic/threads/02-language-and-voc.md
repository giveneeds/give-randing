# Language And VoC

스레드는 전문 용어를 설명하는 채널이 아니라, 독자가 이미 느끼는 답답함을 말로 잡아주는 채널이다.

## 출력 태도

- 독자를 가르치기보다 옆에서 판단 기준을 정리해준다.
- "이걸 당장 하세요"보다 "이 지점부터 보면 판단이 쉬워진다"에 가깝게 쓴다.
- 어려운 용어는 바로 낮춘다. 예: "검색 의도" -> "손님이 검색창에 실제로 넣는 말".
- 추상 조언보다 화면, 문구, 리뷰, 사진, 예약 버튼, 프로필 같은 실제 접점을 말한다.
- 특정 업종이 명확할 때만 "사장님", "원장님" 같은 호칭을 쓴다.
- 업종이 불분명하면 "가게", "병원", "매장", "브랜드", "서비스", "콘텐츠를 올리는 사람"처럼 상황에 맞는 넓은 표현을 쓴다.

## 현장어 치환 원칙

- 전문 용어를 먼저 쓰지 않는다. 필요하면 쉬운 말 뒤에 괄호로만 보조한다.
- `reader_problem`은 보고서 문장이 아니라 현장에서 나오는 고민 문장으로 쓴다.
- "전환율", "퍼널", "ROAS", "세그먼트", "포지셔닝", "브랜딩", "최적화" 같은 말은 그대로 쓰기 전에 생활어로 바꾼다.
- 첫 줄은 지식 전달보다 공감, 의심, 불안, 손해감, 저장 가치를 먼저 건드린다.

좋은 문장은 "마케팅 효율이 낮다"가 아니라 "광고비는 나가는데 전화가 안 온다"에 가깝다.

## 현장어 체크

아래 기준 중 2개 이상이 들어간 문장을 우선한다.

- 돈: 광고비, 대행비, 매출, 예약, 문의, 전화, 상담.
- 시간: 이번 달, 요즘, 3개월째, 매일 올리는데, 주말마다.
- 화면: 플레이스 첫 사진, 리뷰, 블로그 제목, 전화 버튼, 예약 버튼, 메뉴판, 상세페이지.
- 행동: 올렸다, 바꿨다, 맡겼다, 물어봤다, 찍었다, 답글 달았다.
- 감정: 답답하다, 불안하다, 모르겠다, 믿기 어렵다, 손해 보는 것 같다.

## 치환 예시

| 전문 용어 | 스레드용 표현 |
|---|---|
| 전환율이 낮다 | 들어오긴 하는데 문의가 안 온다 |
| 퍼널이 끊긴다 | 손님이 중간에 빠져나간다 |
| ROAS가 안 나온다 | 광고비 쓴 만큼 돌아오는 게 없다 |
| 타깃 세그먼트 | 우리 가게를 고를 사람 |
| 브랜드 포지셔닝 | 손님 머릿속에 남는 한마디 |
| 콘텐츠 최적화 | 사람들이 보고 넘기지 않게 고치는 것 |
| 고객 여정 | 손님이 검색하고 예약하기까지 보는 순서 |
| 신뢰 신호 | 여기 괜찮겠다고 느끼게 하는 단서 |

## 소비자언어

소비자언어는 분석어를 쉽게 바꾼 문장이 아니라, 손님이 검색창에 치거나 옆 사람에게 묻는 말이다.

권장:
- "주차 돼?"
- "지금 열었어?"
- "애 데리고 가도 돼?"
- "예약 되나?"
- "혼자 가도 괜찮아?"
- "사진이랑 실제랑 비슷해?"
- "웨이팅 길어?"

금지에 가까운 초반 표현:
- "~할 때가 많습니다"
- "~하는 단계입니다"
- "~고객을 놓칠 수 있습니다"
- "~가능성이 큽니다"
- "~의도와 어긋납니다"
- "~중심으로 검색합니다"
- "방문 직전 고객", "예약 전환 고객", "구매 퍼널", "검색 의도"

검수 질문:
- 이 문장을 실제 사람이 검색창이나 대화에서 말할 법한가?
- 첫 3줄에 분석어보다 장면/돈/시간/화면/감정이 먼저 보이는가?
- 쉬운 설명문을 소비자언어로 착각하지 않았는가?
